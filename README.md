# Age Calculator (Python)

A simple Python program that calculates your exact age in years, months, and days based on your date of birth.


## What this project does

You enter your birth date, and the program calculates your exact age:

- Years
- Months
- Days

It uses real calendar logic, so the result is accurate.

## Built with

- Python
- datetime module
- dateutil.relativedelta
